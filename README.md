### 4. 💬 단답형 텍스트 기반 MBTI 예측 프로젝트 (자체 분석 프로젝트)

✅ **My Role & Contributions**
- **데이터 전처리 및 분석**: 실제 응답 텍스트를 수집·정제한 뒤, 불용어 제거, 토큰화, 형태소 분석 등 기초 NLP 처리를 수행하고, 응답당 평균 길이·형태 분석을 통해 각 유형별 패턴을 탐색
